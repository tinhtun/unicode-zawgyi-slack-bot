# Unicode Zawgyi Slack Bot
Slack bot to convert unicode to zawgyi font and vice versa

## How to Run
``` bash
$ export SLACK_BOT_TOKEN=xoxb-XXXXXXXXXXXXXXXXXXX
$ pm2 start main.js
```
